Интерфейс

Select Units

Вкладка Select Units

  1. Вкладки Select Units и Configure Game: первая служит для набора армии, вторая — для настройки игры (кроме правил).
  2. Game Options — настройки игровой механики, правил, условий победы и т. д.
  3. Ростер — список своих и вражеских мехов. В зависимости от настроек, вражеские мехи могут быть скрыты частично или полностью.
  4. Управление армией:
    • Add A Combat Unit — добавить юнит в ростер;
    • Create Random Army — сгенерировать армию случайным образом по заданным параметрам;
    • Load Unit List — загрузить ранее сохранённую армию с помощью Save Unit List;
    • Configure Unit — настройка выбранного юнита (то же, что и двойной клик по юниту в ростере);
    • Random Skills — случайным образом задать навыки пилота;
    • Save Unit List — сохранить текущую армию в файл;
    • View Unit — посмотреть характеристики меха;
    • Load Custom BA — по всей видимости, здесь можно создать свой собственный вариант бронепехоты. Никогда не пользовался;
    • View Group — посмотреть описания сразу всех юнитов ростера;
    • Delete Unit — удалить юнит из ростера (клавиша del);
    • Create Squadron — создать эскадрилью;
    • Delete All — удалить все юниты из ростера.
  5. Список игроков.
  6. Чат.
  7. Кнопка, которую должны нажать все игроки из списка 6 (кроме ботов), чтобы начать игру.

Configure Game

Вкладка Configure Game

  1. Текущие характеристики карты: размер, количество мапшитов.
  2. Настройки карты и характеристик планеты.
  3. Настройка стартовых позиций — то, куда игроки смогут выставить своих мехов в начале игры. Доступны варианты: 4 стороны, 4 угла, центр, любой край, любая позиция.
  4. Настройка игрока. Здесь можно выбрать цвет или камуфляж своих мехов, задать бонус к инициативе, объединить игроков в команды, добавить/удалить ботов.
  5. Настройки минных полей.

Окно выбора юнитов

Select Mech…

  1. Фильтр для списка мехов. Можно фильтровать по следующим параметрам:
    • Weight Class — вес (лёгкий, средний, тяжёлый, штурмовой);
    • Type — технологическая база (Внутренняя Сфера, Кланы);
    • Unit Type — тип юнита (мех, танк, пехота);
    • Sort — по какому параметру сортировать список юнитов.
  2. Display — какие колонки отображать в списке мехов.
  3. Show Advanced Search — показать расширенный поиск.
  4. Список юнитов.
  5. Подробное описание выбранного в списке юнита.
    • Select — добавить выбранный мех в ростер;
    • Select & Close — добавить выбранный мех в ростер и выйти;
    • Close — закрыть окно выбора юнита.
  7. Игрок, которому добавляется выбранный мех. Здесь можно указать себя или бота.

Mech Configure

  • Name — имя пилота;
  • Gunnery — навык стрельбы;
  • Piloting — навык пилотирования;
  • Disable Automatic Ejection — отключить автоматическое катапультирование (полезно, если есть CASE);
  • Deployment Round — в каком раунде юнит будет выставлен на карту.
  • Далее идёт список боеприпасов с возможностью выбора конкретных типов. Галочка Dump This Ammunition позволяет сбросить боеприпасы еще до начала игры (например, бывает полезно избавиться от патронов к пулемёту).

Основной игровой экран

  1. Игровая карта. Жёлтым отмечены гексы, куда мне разрешено выставлять мехов.
  2. Список моих мехов. Шкалы показывают количество брони, внутренних структур и нагрев.
  3. Кнопки (относятся к выбранному в данный момент юниту):
    • Next Unit — выбрать следующий юнит;
    • Turn — повернуть юнит на месте (можно зажать shift и кликнуть лкм в нужном направлении);
    • Load — погрузить пехоту в транспортное средство;
    • Unload — выгрузить пехоту;
    • Remove Unit — избавиться от юнита (он не будет участвовать в сражении).
  4. Здесь написано чья сейчас очередь ходить.

В фазу движения добавляются дополнительные кнопки. Поскольку на экран они все не помещаются. Кнопка More переключает наборы. А кнопки следующие:

  • Walk — идти вперёд (полезно, если сначала пятился);
  • Jump — прыгать;
  • Back Up — идти задом;
  • Get Up / Go Prone — встать/лечь;
  • Charge — таран (физатака);
  • D.F.A. — Death From Above — смерть сверху (физатака);
  • Eject — катапультирование;
  • Flee — сбежать с поля боя (только находясь у края карты).

Кнопки фазы оружейной атаки:

  • Fire — стрелять выбранным оружием по выбранной цели;
  • Skip — перейти к следующему оружию в списке;
  • Next Target — следующая цель;
  • Twist — поворот туловища (то же, что и shift + лкм);
  • Mode — сменить режим стрельбы текущего оружия;
  • Flip Arms — вывернуть руки назад;
  • Find Club — найти дубину (в лесу), чтобы использовать в качестве оружия;

Текущие характеристики меха

Вкладка General

Вкладка General

  • Player — игрок, который управляет данным юнитом;
  • Team — команда, в которой играет игрок, управляющий данным юнитом;
  • Status — статус (стоит/лежит);
  • Weight — вес;
  • Current BV — текущее BV (оно уменьшается из-за повреждений);
  • Movement — очки движения для ходьбы, бега и прыжка;
  • Currently — каким типом движения и на сколько переместился юнит;
  • Heat — текущий нагрев и ёмкость радиаторов;
  • Pilot — имя пилота, повреждения им полученные и его навыки;
  • Auto Eject — включено ли автоматическое катапультирование;
  • Elevation — высота (для VTOL‘ов).

Вкладка Armor

Вкладка Armor

  • Front armor — передняя броня;
  • Rear armor — задняя броня;
  • Internal Structure — внутренние структуры;
  • Heat — шкала нагрева.

Вкладка Systems

Вкладка Systems

  • Location — список локаций;
  • Slot — слоты выбранной локации, звёздочками помечены слоты получившие критические повреждения;
  • Mode — режим работы выбранного слота;
  • Dump — сбросить боеприпасы из выбранного слота.

Вкладка Weapon

Вкладка Weapon

На этой вкладке показан список оружия и подробная информация по текущему выбранному оружию: типа боеприпаса, название, теплоотдача, урон, дистанции стрельбы. Так же показана информация о текущей цели: название, расстояние, вероятность попадания с подробным списком модификаторов.

Вкладка Extras

Вкладка Extras

  • Current Sensors — текущий режим работы сенсоров;
  • Affected By — под действием каких эффектов находится юнит;
  • Unused Space — свободное пространство (для переноски чего-либо);
  • Carrying — груз;
  • # of active Heat Sinks — число активных радиаторов;
  • Heat Effects — эффекты от перегрева;
  • Targeting System — режим работы системы прицеливания.
